区分大小写 ECMAScript中 所有变量 函数名 属性的名字 操作符都是区分大小写的； text与Text是两种不同的标识符
所谓标识符 就是指变量 函数 属性的名字 或者 函数的参数； 标识符 规则： 标识符 组成 必须是开头：以 字母 下划线 到了富$开头 不能以@ 和数字开头 其他组成就是 数字； 不能把关键字 保留字 true false null作为标识符
ECMA中注释 单行注视//两个反斜杠;多行注释 注释的目的： 对代码进行说明 ；对代码进行调试 都会用到注释
100 数字字面量 '李彦辉' 字符串字面量 false 布尔值字面量 /js/gi 正则表达式字面量 null 对象字面量 {x:1,y:2} 对象字面量 （用花括号 键值对方式 书写）
ECMA的关键字 关键字 是程序中 一组有特定用途的关键字 一半用于控制语句的开始或结束 或者用于执行特定的操作 保留字 目前在javascript中用不到 但以后可能要用到 所以在编码 时也不能使用这些作为变量名 与 函数名
声明变量 ；变量初始化（赋值）；变量的调用 输出变量的值 var box =100; alert(box);以弹窗的方式输出
ECMA中的变量是弱类型的 如果变量声明 没有被初始化 则script默认给初始值 undefined； alert(age); age就没有初始化 如果 age is not defred 不存在 和 未定义 是两码事 ECMA 所谓变量初始化以后可以再次改变的量 ECMA 可以再重新赋值时改变变量的数据类型 弱类型的语言 这样会为程序的维护带来很大难度 alert(typeof(box)); ECMA 中声明变量 不需要使用var的情况 这个和变量的作用域有很大关系； ECMA中 当你每条语句都在不同行时要加分号 作语句结束符； 可以使用一条语句 定义多个变量 var box='李彦辉'，age=28,height＝178；
